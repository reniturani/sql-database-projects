# sql-database-projects
Repositorio personal de proyectos académicos realizados en SQL / MySQL, orientado a la práctica del diseño y desarrollo de bases de datos relacionales.
Este repositorio forma parte de mi perfil como estudiante de programación y tiene como objetivo mostrar proyectos integradores desarrollados durante mi formación, aplicando conceptos clave de bases de datos.

## Objetivo del repositorio
Practicar el modelado de bases de datos relacionales
Aplicar buenas prácticas de SQL
Implementar lógica de negocio en la base de datos
Contar con proyectos documentados para el perfil académico y laboral

## Tecnologías utilizadas
MySQL
SQL estándar
